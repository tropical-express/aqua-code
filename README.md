# Aqua Code

AquaCode is a beginner-friendly desktop code editor with an aquarium-inspired UI.

## Features (alpha)
- Desktop code editor
- Ocean-themed background
- Bubble animations
- Simple file editing
- Built with Electron

## Goal
A calm, visual coding environment that feels more fun and less intimidating than traditional editors.

## Status
Early alpha build (v0.1.0)

Expect bugs and missing features.

```bash
pnpm install
